# 🎮 بوت Discord لحماية سيرفر Minecraft

## المميزات

### 1. حماية ضد التخريب
- **مضاد السبام**: يكشف ويمنع السبام تلقائياً
- **طرد الأعضاء المخربين**: أوامر لإدارة الأعضاء
- **تقييد الأعضاء**: كتم مؤقت للأعضاء المخلين

### 2. نظام اللوقات الشامل
- لوقات الرسائل (حذف، تعديل)
- لوقات الأعضاء (دخول، خروج، تحديثات)
- لوقات القنوات والرتب
- لوقات الموديريشن
- **لوقات خاصة للإدارة فقط** (مخفية)

### 3. عرض قائمة الأعضاء
- قائمة كاملة بأعضاء السيرفر
- حالة كل عضو (متصل، بعيد، غير متصل)
- الرتب لكل عضو
- البحث عن أعضاء

### 4. الحركة التلقائية
- حركة تلقائية كل 10 ثواني
- تمنع خروج البوت من السيرفر

## التثبيت

### 1. المتطلبات
```bash
pip install -r requirements.txt
```

### 2. إعداد البوت في Discord Developer Portal
1. اذهب إلى: https://discord.com/developers/applications
2. أنشئ تطبيق جديد
3. أضف بوت
4. انسخ الرمز (Token)

### 3. إعداد config.json
```json
{
    "token": "YOUR_BOT_TOKEN_HERE",
    "prefix": "!",
    "admin_role_id": 123456789012345678,
    "log_channel_id": 123456789012345678,
    "private_channel_id": 123456789012345678,
    "server_id": 123456789012345678,
    "main_guild_ip": "your.server.ip",
    "main_guild_port": 25565,
    "anti_spam_settings": {
        "max_messages": 5,
        "time_window": 10,
        "mute_duration": 60
    },
    "auto_wiggle": {
        "enabled": true,
        "interval": 10
    }
}
```

### 4. إنشاء القنوات
- **قناة اللوقات الخاصة**: أنشئ قناة خاصة للمشرفين فقط
- **قناة اللوقات العامة**: قناة عامة لللوقات الأساسية

### 5. تشغيل البوت
```bash
python bot.py
```

## الأوامر

| الأمر | الوصف |
|-------|-------|
| `!القائمة` | عرض قائمة الأعضاء |
| `!الة` | عرض معلومات السيرفر |
| `!مساعدة` | عرض قائمة الأوامر |
| `!البحث <اسم>` | البحث عن عضو |
| `!حالة` | عرض حالة البوت |
| `!سجل` | عرض السجلات (للمشرفين) |
| `!مسح <عدد>` | مسح رسائل |
| `!طرد <عضو>` | طرد عضو |
| `!حظر <عضو>` | حظر عضو |
| `!تقييد <عضو>` | تقييد عضو |
| `!الغاء_التقييد <عضو>` | إلغاء تقييد عضو |

## الصلاحيات المطلوبة للبوت
- إدارة الرسائل (Manage Messages)
- طرد الأعضاء (Kick Members)
- حظر الأعضاء (Ban Members)
- إدارة الرتب (Manage Roles)
- إدارة القنوات (Manage Channels)
- عرض سجلات المراجعة (View Audit Log)

## ملاحظات
- تأكد من أن البوت لديه الرتب المناسبة فوق رتب الأعضاء
- القناة الخاصة يجب أن تكون visible للمشرفين فقط
- يمكنك تعديل إعدادات مضاد السبام في config.json
